## 主要功能：
支持 多级别日志消息
支持 同步和异步发送（性能差异大）
支持 写入到控制台/文件/指定文件
支持 滚动日志（文件）
支持 多线程并发写日志
支持 扩展不同的日志落地目标地


## 模块设计：

# 全局控制模块（logger）
1. 一个全局变量级别，比它等级小的消息屏蔽
3. 通过格式、级别模块获取整体日志消息
4. 通过发送模块、sink模块将消息落地

# 级别模块（level）
1. 一个级别枚举（注意等级int是有意义的，要正确排序）
2. 字符级别 和 枚举级别 的相互转换

# 信息模块（message）
1. 信息对象，组成：级别+时间+文件名+线程号+行号+正文

# 格式化模块（formatter）
1. 将信息格式化并返回

# 落地模块（sink，基类）
1. 确定落地地点
2. 落地动作

# 滚动模块（rollsink）
1. 继承sink
2. 对于sink增加落地动作前后处理（检查时间/空间，判断创建/删除/写入目标）

# 发送模块 （sync/async，同步发送&异步发送）
1. 同步：直接调用sink模块
2. 异步：日志写入内部缓冲区，一个后台线程调用sink管理缓冲区内容落地
3. 一个内部缓冲区，用于缓存异步发送的信息

# 扩展模块
1. 特殊的sink
2. 定位外部的落地点接口
3. 特殊的落地动作

**多线程并发**：保证sink的线程安全
